# Cyclistic Bike Share — Análise Exploratória de Dados

Este projeto apresenta uma análise exploratória dos dados do sistema de compartilhamento de bicicletas Cyclistic, desenvolvida como trabalho final do curso Google Data Analytics Professional Certificate.

## Objetivo

A análise tem como principal objetivo compreender o comportamento dos usuários ao longo do período de janeiro a dezembro de 2025, com foco na comparação entre:

Usuários casuais (casual riders)
Membros anuais (annual members)

A partir dessa investigação, busca-se identificar padrões de uso, diferenças entre perfis e gerar insights estratégicos que possam auxiliar na conversão de usuários ocasionais em membros anuais.

## Principais etapas da análise
- Limpeza e tratamento dos dados
- Análise exploratória (EDA)
- Identificação de padrões de uso (tempo, frequência, sazonalidade)
- Comparação entre tipos de usuários
- Geração de insights e recomendações estratégicas
